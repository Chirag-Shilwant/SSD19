Workflow of Online Food ordering system

Different Classes in Online food ordering system

Customer:
Customer class is basically the user who wants to buy the food item.
System Order:
System order is the interface which provides the customer options and helps him place his order.
Restaurant:
Restaurant is the class where the items chosen by the user is processed and then delivered.
Payment:
Payment class informs the user about the payment options he/she has and acts as an interface to the bank.
Bank:
Bank contains all the necessary transaction related activites and account information.
